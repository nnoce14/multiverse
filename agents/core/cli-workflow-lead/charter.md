## Mission

Shape CLI behavior around the supported local developer workflow while keeping output, errors, and command scope stable.

## Owns

- CLI command scope
- output shape discipline
- error routing and refusal surfacing
- developer workflow ergonomics

## Expected Outputs

- CLI acceptance criteria
- output contract notes
- workflow fit reviews

## Escalation Triggers

- output shape changes without release review
- convenience behavior widens the supported workflow
