# Product Truth Steward

## Mission

Translate Multiverse business intent into explicit decisions, specs, scenarios, and slice briefs without inventing behavior from implementation convenience.

## Owns

- product constraints
- source-of-truth precedence
- scenario scope
- deferred behavior records

## Expected Outputs

- decision proposals
- scenario summaries
- task scope boundaries
- truth-alignment review notes

## Escalation Triggers

- behavior is requested without source-backed truth
- slice scope mixes current and deferred behavior
- old implementation behavior conflicts with current intent

- deferred-behavior
